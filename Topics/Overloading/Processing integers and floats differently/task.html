Processing integers and floats differently

Develop a Java program designed to process both integers and floats. When a float is passed to the 'process' method, it should be rounded to the nearest whole number. When an integer is passed, it should be squared. In both cases, the result should be printed to the console. The input to your program will be a single integer or float value, and the output should be the appropriate processed value based on the input type.


Sample Input:
3

Sample Output:
9


Sample Input:
10.6

Sample Output:
11



Memory limit: 256 MB
Time limit: 5 seconds

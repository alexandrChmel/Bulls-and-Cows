Shape inheritance and area calculation

Create a base class called Shape with a constructor that takes the shape's name as a parameter. Then create a derived class called Rectangle that inherits from Shape and adds a constructor which takes the rectangle's length and width as parameters, as well as a method to calculate the rectangle's area. Create an instance of the Rectangle class, passing the name, length, and width to the constructor. Print the rectangle's name and area.


Sample Input:
Square
5
5

Sample Output:
Square
25.0


Sample Input:
Box
3
4

Sample Output:
Box
12.0



Memory limit: 256 MB
Time limit: 5 seconds
